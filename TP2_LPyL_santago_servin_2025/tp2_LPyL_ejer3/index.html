las paginas web

las paginas web sonel principal recurso de la web, las primeras paginas web fueron las estaticas que surguieron en la web 1.0, luego vinieron las dinamicas en la web 1.5.

las paginas web son interpretadas por el navegador web, cuando son descargadas al dispositivo del usuario, las paginas estan conformadas por HTML, CSS, un archivo de ejecucion a nivel de usuario (.js) y un archivo de ejecucion a nivel de servidor (.php).

dependiendo en donde los scrips de procesamieto se pueden dividir en dos grupos

  • paginas estaticas
  • paginas dinamicas:
    • paginas dinamicas de usuario
    • paginas dinamicas de servidor
    • paginas dinamicas mixtas

paginas estaticas

las paginas estaticas fueron las primeras en aparecer mostrando un mismo contenido a todos los usuarios, eran creadas por instituciones son paginas web que se ejecutan por el lado del usuario

paginas dinamicas

las paginas dinamicas pueden procesar datos ingresados por el usuario, pudinedo mostrar contenido personalizado en relacion a sus preferencias

creacion de paginas web dinamicas

  1. elicitacion de requerimeitos: se investigan las funcionalidades necesarias para la pagian web, (entrevistas, encuestas, cuestionarios, etc)
  2. se analizan los requerimientos (especificacion de requerimeitos): formando modelos para un proceso de desarrollo
  3. diseño de la pagina web: se pasan esos modelos del analisis a modelos mas especificos del diseño. teniendo un menor grado de abstraccion
  4. implementacion: se traducen a (html, css, javascript, php) los diseños ya armados
  5. test y validacion: se testean las funcionalidades para comprobar su funcionalidad y se validan con el cliente par acommprobar su efectividad
  6. mantenimiento: se agregan nuevas funcionalidades, se expanden lsa existentes, se corrigen errores (llevan mucho mas mantenimiento que un programa de escritorio)
